Những tính năng nổi bật của ES6 – Viết JavaScript hiện đại và gọn gàng hơn

🕓 Đăng ngày 15/10/2025

⚙️ “ES6 là bước ngoặt giúp JavaScript trở nên mạnh mẽ và dễ đọc hơn bao giờ hết.”

Phiên bản ECMAScript 2015 (ES6) mang đến hàng loạt tính năng giúp lập trình JavaScript hiện đại, sạch và dễ bảo trì hơn.

🚀 1. let và const – Khai báo biến chuẩn hiện đại

let name = "Trang";
const PI = 3.14;

name = "Huyền Trang"; // ✅ OK
PI = 3.14159; // ❌ Lỗi – không thể gán lại const

📘 Ghi nhớ:

  • let: dùng cho biến có thể thay đổi.
  • const: dùng cho giá trị cố định.
  • Cả hai đều có phạm vi block-scope (chỉ tồn tại trong {}), khác với var.

🧩 2. Arrow Function – Cú pháp ngắn gọn hơn

// Cũ
function greet(name) {
  return "Xin chào, " + name;
}

// ES6
const greet = (name) => `Xin chào, ${name}!`;

📎 Lợi ích:

  • Ngắn gọn, dễ đọc.
  • Tự động kế thừa this từ phạm vi cha → rất hữu ích trong callback.

💬 3. Template Literals – Nội suy chuỗi dễ dàng

const user = "Trang";
const age = 22;

console.log(`Xin chào ${user}, bạn ${age} tuổi.`);

✨ Có thể xuống dòng tự nhiên mà không cần \n:

const message = `Học ES6 thật thú vị!
JavaScript ngày càng mạnh mẽ hơn.`;

⚙️ 4. Destructuring – Tách giá trị dễ dàng

const user = { name: "Trang", age: 22, city: "Hà Nội" };

const { name, city } = user;
console.log(name, city); // Trang Hà Nội

const arr = [10, 20, 30];
const [a, b] = arr;
console.log(a, b); // 10 20

📚 Ứng dụng: Lấy giá trị nhanh trong object/array mà không cần gọi từng phần tử.

🌐 5. Spread và Rest Operator (...)

Spread (...) – Dải giá trị

const arr1 = [1, 2];
const arr2 = [...arr1, 3, 4];
console.log(arr2); // [1, 2, 3, 4]

Rest (...) – Gom giá trị

function sum(...nums) {
  return nums.reduce((a, b) => a + b);
}
console.log(sum(1, 2, 3)); // 6

🧠 Ghi nhớ: Spread để “dải”, Rest để “gom”.

🧱 6. Class – Cú pháp hướng đối tượng hiện đại

class Student {
  constructor(name, age) {
    this.name = name;
    this.age = age;
  }

  greet() {
    console.log(`Xin chào, tôi là ${this.name}`);
  }
}

const s = new Student("Trang", 22);
s.greet();

✅ Tính năng ES6 Class: hỗ trợ constructor, extends, super, và kế thừa như các ngôn ngữ OOP khác.

📦 7. Module – Tổ chức mã nguồn dễ quản lý

👉 math.js

export function add(a, b) {
  return a + b;
}
export const PI = 3.14;

👉 main.js

import { add, PI } from "./math.js";
console.log(add(2, 3), PI);

📎 Lợi ích: chia code thành module nhỏ, dễ bảo trì, tránh xung đột biến toàn cục.

⚡ 8. Default Parameters

function greet(name = "bạn") {
  console.log(`Xin chào, ${name}!`);
}

greet(); // Xin chào, bạn!
greet("Trang"); // Xin chào, Trang!

📘 Giúp tránh lỗi “undefined” khi không truyền đủ tham số.

🔁 9. Promise – Xử lý bất đồng bộ dễ hơn Callback

const getData = new Promise((resolve, reject) => {
  setTimeout(() => resolve("Dữ liệu đã sẵn sàng!"), 1000);
});

getData.then(console.log).catch(console.error);

✅ Thay thế callback hell bằng cú pháp gọn gàng, dễ đọc.

🧾 10. Tổng kết ES6

Tính năng Mô tả
let, const Khai báo biến an toàn hơn
Arrow Function Cú pháp ngắn, kế thừa this
Template Literals Chuỗi nội suy tiện lợi
Destructuring Lấy giá trị dễ dàng
Spread / Rest Dải và gom giá trị
Class Lập trình OOP
Module Chia nhỏ mã nguồn
Promise Xử lý bất đồng bộ hiện đại

🎯 Kết luận

  • ES6 đã đưa JavaScript trở thành ngôn ngữ hiện đại, mạnh mẽ và dễ học hơn.
  • Nắm vững các tính năng này giúp bạn viết code ngắn gọn, hiệu quả, và ít lỗi hơn.
  • Đây là nền tảng để học tiếp React, Node.js, và TypeScript.

💡 “Nếu bạn chưa dùng ES6, bạn đang viết JavaScript như năm 2010.”
